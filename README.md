# Async URL Shortener Service

Асинхронный сервис для сокращения URL-адресов на основе FastAPI с поддержкой асинхронных операций.

## Описание

Современный асинхронный сервис для сокращения URL-адресов, разработанный с использованием FastAPI и асинхронных возможностей Python. Приложение обеспечивает высокую производительность за счет асинхронной обработки запросов и асинхронного взаимодействия с базой данных.

## Особенности

- **Полностью асинхронный** - все обработчики и сервисные функции используют async/await
- Высокая производительность благодаря асинхронной архитектуре FastAPI
- Поддержка одновременной обработки множества запросов
- Асинхронное взаимодействие с базой данных через SQLAlchemy с асинхронными движками
- RESTful API для интеграции с внешними системами

## Технологии

- **FastAPI** - современный веб-фреймворк с поддержкой асинхронности
- **SQLAlchemy (асинхронный)** - ORM с поддержкой async/await
- **MySQL** - реляционная база данных
- **Uvicorn** - ASGI-сервер для запуска асинхронного приложения
